Как заполнять отчёт в Telegram

Гайд для дизайн-команды Tranio

Каждый день пишешь в свою группу "Трекер" что сделал за день.

Формат

Каждая задача — по шаблону:

подразделение: что делал — сколько времени — кто отвечает за задачу

Подразделение

Пиши в начале каждой задачи:

  • broker — задача для Tranio Brokerage
  • capital — задача для Tranio Capital

Время

Пиши как удобно:

  • Часы: 2ч, 1.5ч, 0.5ч
  • Минуты: 30м, 45м, 15м
  • Комбинация: 2ч 30м

Заказчик

Пиши имя того, кто отвечает за задачу:

  • Лена, Соня, Женя, Алеся, Юля, Влада, Катя, Сергей, Ринат, Марина — Brokerage
  • Георгий, Ира, Макс, Слава, Илья — Capital
  • аффилиаты — если задача для партнёрки (даже если пришла от Лены или Сони)

Задача

Пример новой простой задачи

broker: Презентация для конференции в Дубае — 3ч — Лена

Пример правок в простой задаче

broker: Правки презентации для конференции в Дубае — 1ч — Лена

Пример задачи для партнёрки

broker: Презентация для партнёрской программы — 2ч — аффилиаты

Пример задачи как части большого проекта

Например, нам ставят большую задачу — "Новая программа ВНЖ Греции", которая состоит из нескольких подзадач. В этом случае в простой задаче достаточно упомянуть название большой задачи:

broker: Макет лендинга Новая программа ВНЖ Греции — 3ч — Лена broker: Креативы для фейсбука Новая программа ВНЖ Греции — 2ч — Лена broker: Тизеры для инстаграм Новая программа ВНЖ Греции — 1ч — Лена

Правки — так же:

broker: Правки лендинга Новая программа ВНЖ Греции — 1.5ч — Лена

Если задача вернулась из бэклога — это доработка:

broker: Доработка лендинга Новая программа ВНЖ Греции — 2ч — Лена

Срочные задачи, задержки фидбэка и влетуны

Если задача прилетела срочно или вы долго ждали обратную связь или прилетел влетун мимо недельного скоупа — напиши об этом в скобках:

broker: Дизайн тизера — 30м — Соня (влетун)
broker: Правки презентации — 2ч — Лена (ждали фидбэк 5 дней, срочно в конце дня)
capital: Переделка инфографики — 3ч — Ира (срочно, попросили за час до конца дня)
broker: Доработка лендинга — 1.5ч — Сергей (фидбэк пришёл через неделю)

Это помогает нам видеть реальную картину и понимать откуда берутся авралы.

Выходные и отпуск

Если берёшь дей-офф, отпуск или больничный — просто напиши в группу:

дей-офф
отпуск 01.04-05.04
больничный

Примеры отчета за день

Можно писать двумя способами — как удобнее:

Отдельными сообщениями по ходу дня

broker: Презентация для конференции — 3ч — Лена
broker: Правки лендинга недвижимости — 1.5ч — Сергей
capital: Инфографика для отчёта инвесторам — 2ч — Ира

Одним сообщением в конце дня

broker: Презентация для конференции — 3ч — Лена broker: Правки лендинга недвижимости — 1.5ч — Сергей capital: Инфографика для отчёта инвесторам — 2ч — Ира broker: Тизеры для инстаграм — 45м — Соня

Если ошиблись в сообщении или нужно дополнить — сообщения можно редактировать до 19:59. Бот забирает сообщения в обработку в 20:00.

💬 Если что-то непонятно — спроси Андрея.
